A wrapper around an i.MX RT processor pad, supporting simple I/O

Use new with a pad from the iomuxc module. All GPIOs start in the input state. Use output to become an output pin.

use imxrt_async_hal as hal;
use hal::gpio::GPIO;

let pads = hal::iomuxc::new(hal::ral::iomuxc::IOMUXC::take().unwrap());
let mut led = GPIO::new(pads.b0.p03);
assert!(!led.is_set());
let mut led = led.output();
led.set();

When using a GPIO as an input, you can wait for transitions using wait_for.
