[−][src]Module imxrt_async_hal::iomuxc

Pad multiplexing and configuration

The iomuxc module is a re-export of the imxrt-iomuxc crate. It combines the i.MX RT processor-specific components with the imxrt-iomuxc general API. It then adds a safe function, take, which lets you convert the RAL's iomuxc::Instance into all of the processor Pads.

use imxrt_async_hal as hal;
use hal::{ral::iomuxc::IOMUXC, iomuxc};

let pads = iomuxc::new(IOMUXC::take().unwrap());

Pads can then be used in peripheral-specific APIs.
