Direct Memory Access (DMA) for async I/O

DMA Channels power some asynchronous I/O operations. Use channels to acquire all of the processor's DMA channels. Then, use the Channels in APIs that require them. The implementation handles DMA receive and transfer operations, and ensures that the lifetime of your buffers is correct.
